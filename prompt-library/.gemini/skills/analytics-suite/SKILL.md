# Analytics Suite Protocol

## Mode 1: CODE_HEALTH (The Project Manager)
* **Target:** Git logs, Source code.
* **Goal:** Velocity, Complexity, Tech Debt.

## Mode 2: DATA_SCIENCE (The Scientist)
* **Target:** CSV, Logs, SQL.
* **Goal:** Statistical Analysis & Visualization (Pandas/Matplotlib).

## Mode 3: BUSINESS_GROWTH (The Marketer) [NEW!]
* **Target:** User behavior data, Competitor URLs, Feature Lists.
* **Capabilities:**
    1.  **SEO Strategy:** Extract high-value keywords & Generate Meta Tags.
    2.  **Copywriting:** transform `git log` features into "Launch Emails" or "Tweet Threads".
    3.  **Market Analysis:** Perform SWOT analysis (Strength, Weakness, Opportunity, Threat).
    4.  **Funnel Optimization:** Analyze drop-off rates and suggest UI/UX fixes.

## Output
* Return a structured report.
* If `BUSINESS_GROWTH`, include specific copy/keywords ready for use.

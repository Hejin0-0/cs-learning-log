# 🚀 Prompt Upgrade Repository

이 저장소는 프롬프트의 성능을 개선하고 최적화하는 모든 과정을 기록하는 **프롬프트 정제소(Prompt Refinery)**입니다. 
각 단계별 변경 사항과 테스트 결과를 체계적으로 관리하여 최적의 프롬프트 엔진을 구축하는 것을 목표로 합니다.

---

## 📂 폴더 구조 및 규칙

모든 업그레이드 기록은 하나의 폴더 내에 독립된 파일로 관리됩니다.

### 1. 파일 명명 규칙 (Naming Convention)
파일을 한눈에 파악하고 정렬하기 위해 아래 형식을 권장합니다.
- **형식**: `YYMMDD_v[버전]_[주요내용].[확장자]`
- **예시**: `260205_v1.2_Add_MoE_Logic.md`

### 2. 권장 확장자
- **.md (Markdown)**: 빠른 기록 및 코드 블록 작성용 (추천)
- **.docx / .pdf**: 외부 공유 및 정식 보고서용
